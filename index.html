Projects

Data Analysis Model

핵심이 되는 raw data는 사출 공정이 끝난 직후 제품을 촬영한 열화상 이미지의 온도 데이터와 양/불량 판정의 기준이 될 라벨 데이터 역할을 할, 검수 담당자가 수기 기록한 완제품의 두께 정보로 이루어져 있습니다. 이 1차 가공 데이터에서 열화상 이미지 데이터에는 관심영역을 설정, 라벨 파일에는 양/불량을 tagging해 2차 가공, 이렇게 처리된 데이터를 학습시킨 모델에 임의 shuffle된 dataset을 input하면 제품의 양/불량 여부를 알 수 있게 됩니다.

Web Application

WebtoB와 Jeus에 올라갈 이 애플리케이션은 관리자 로그인 기능과 불량품 정보를 알려주는 기능(관리자 처리)을 포함합니다. 로그인 UI는 ID값을 기본키로 하는 '관리자DB'를 통해 요청을 처리하고, 불량품의 정보가 기록된 로그 파일을 생성하여 관리자 이메일로 전송시켜주는 기능은 관리자 처리 UI에서 확인토록 하였습니다. 또 관리자 처리 UI에서는 이 로그정보를 토대로 불량품 데이터만을 폐기(삭제)시킬 수도 있습니다.
